## Intent (What we want to achieve)
- Build confidence using digital 3D tools to design purposeful products.  
- Apply measurement, symmetry and simple parameters to improve designs.

- **Weekly 90-minute sessions** throughout the Spring term.  
- Each week introduces a **new design idea** inside a **small mini-project**.  
- Activities are tiered so that beginners can follow a guided path while confident learners extend and personalise.  
- Plans may adapt depending on time, resources or the group‚Äôs interests.
